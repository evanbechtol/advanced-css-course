# 3 Pillars of Writing Good HTML and CSS

1. Responsive Design
2. Maintainable and Scalable
3. Web Performance

### Responsive Design
    - Fluid layouts
    - Media queries
    - Responsive images
    - Correct units
    - Desktop-first vs. Mobile-first 

### Maintainable and Scalable
    - Clean
    - Easy-to-understand
    - Encourages growth
    - Reusable
    - Good file structure and organization
    - Good naming conventions
    - Good HTML structure

## Web Performance
    - Less HTTP requests
    - Less code
    - Compress code
    - Less images
    - Compress images
